# 🧮 Calculator

A simple, responsive calculator built using **HTML, CSS, and JavaScript**. Performs basic arithmetic operations with a clean, user-friendly interface.

## ✨ Features

- Addition, subtraction, multiplication, and division
- Clear (C) and delete functionality
- Responsive design — works on desktop and mobile
- Clean UI with keyboard-style button layout

## 🛠️ Tech Stack

- **HTML5** – structure
- **CSS3** – styling and layout
- **JavaScript** – calculation logic and DOM manipulation

## 🚀 How to Run

1. Clone the repository
   ```bash
   git clone https://github.com/Nikitaagg01/Calculator.git
   ```
2. Open `index.html` in your browser — no build steps or dependencies required
